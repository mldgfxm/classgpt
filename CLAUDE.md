# 任务总结规则

每次完成一个任务后，必须输出以下格式的总结：

## 📋 任务总结

### ✅ 完成的任务
- [任务1]: [简短描述]
- [任务2]: [简短描述]

### ❌ 遇到的问题
- [问题1]: [错误信息]
- [问题2]: [错误信息]

### 🔧 解决方案
- [问题1] → [解决方法]
- [问题2] → [解决方法]

### 📁 修改的文件
- `path/to/file1.ts` — [修改内容]
- `path/to/file2.ts` — [修改内容]

### 📌 待办事项
- [ ] [下一步操作]（如果没有则写"无"）

**注意**：这个总结必须在每次任务完成后立即输出，不需要用户提醒。
